Revwalk#walk Sync

var commit = revwalk.walk(oid, callback);

Walk the history from the given oid. The callback is invoked for each commit; When the walk is over, the callback is invoked with (null, null).

Parameters Type
oid Oid
callback Function
Returns
Commit
